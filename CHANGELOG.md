## [0.1.0] - 2026-03-21

### Added
- Initial release
- Word-boundary-aware text wrapping with configurable width
- Indentation support with uniform and hanging indent
- ANSI escape code-safe width calculation
- Hard wrap for words exceeding line width
- Text truncation with configurable omission string
